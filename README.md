# goit-algo-hw-08

## Завдання 1 — мінімальний бюджет з'єднання кабелів

- Реалізовано функцію `min_connection_cost` (`task-1.py`), що використовує мін-купу для для greedy об'єднання двох найкоротших кабелів на кожному кроці.
- Це класичний підхід à la алгоритм Гаффмана, який гарантує мінімальну суму проміжних з'єднань.
- Скрипт містить приклади використання в блоці `__main__`.

### Запуск

```bash
cd goit-algo-hw-08
python3 task-1.py
```
